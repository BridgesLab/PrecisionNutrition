# Summary Statistics from PheWeb

These are not availabe in teh reposotitory as they are only available to University of Michigan researchers.  These data were from the dataset MGI BioVU Labs Meta: 70 meta-analyzed phenotypes for 800,000 variants.  They were downloaded on 2025-09-26 via PheWeb version 1.3.15.  The URL was [https://pheweb.org/MGI-BioVU/](https://pheweb.org/MGI-BioVU/)

## Files

There are files for each of the lipid and calcium traits.

- phenocode-Ca.tsv.gz for Ca: Calcium, blood electrolyte
- phenocode-Chol.tsv.gz for Chol: Total cholesterol
- phenocode-HDL.tsv.gz for HDL: High density lipoprotein cholesterol
- phenocode-LDL.tsv.gz for LDL: Low density lipoprotein cholesterol
- phenocode-Trigs.tsv.gz for Trigs: Triglycerides, tested as part of cholesterol panels
