Plink2 format

Plink2 uses the binary pgen/pvar/psam formating. These are binaries that can be generated by --make pgen commands which start from a bgen file, a vcf file or a plink1 set of bed/bim/fam files.

Plink BED file

To make a bed file used the genio package to create bed/bim/fam files

Need a matrix of are m loci and n individuals.

  allele.counts %>%

  group_by(Locus) %>%
  filter(Allele.Count == max(Allele.Count)) %>%

annotated.df <-
  allele.counts %>%
  ungroup %>%
  filter(Allele.Count>836*0.05) %>% #only alleles with greater than 5% occurance
  left_join(major.alleles, by="Locus", suffix = c("_all","_major")) %>%
  filter(Allele_all!=Allele_major) %>% #remove SNPS marked with the common allele
  mutate(ID = paste(Locus,Allele_major,Allele_all,sep="_")) %>% #minor allele first to match bimbam format
  filter(Locus != 'lue') %>%
  select(-Allele_major,-Allele_all,-Allele.Count,-Locus) %>% #dont need allele names or counts anymore
  distinct(ID,.keep_all = T) %>% #need unique row names
  column_to_rownames("ID") %>%
  as.matrix
